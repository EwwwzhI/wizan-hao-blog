# EwwwzhI 个人站项目解析

这份文档说明当前简洁版仓库的结构、内容链路和日常维护方式。

## 项目定位

这是一个基于 Astro 6 的个人档案站，保留三个核心内容入口：

| 页面 | 路由 | 数据来源 |
| --- | --- | --- |
| 首页 | `/` | `src/content/home/index.md` |
| 博客列表 | `/blogs/` | `src/content/blogs/**/*.md(x)` |
| 博客详情 | `/blogs/[slug]/` | `src/content/blogs/**/*.md(x)` |
| 项目页 | `/projects/` | `src/content/projects/data.json` |

站点继续保留 Pagefind 搜索、明暗主题、TOC、OG image 和 sitemap。

## 目录结构

```text
src/
  components/
    views/        # RenderPage, RenderPost, ListView, GroupView
    widgets/      # LogoButton, SearchSwitch, ThemeSwitch, BackLink
    nav/          # 顶部导航
  content/
    home/         # 首页正文
    blogs/        # 博客 Markdown / MDX
    projects/     # 项目 JSON 数据
  layouts/        # 页面外壳
  pages/          # 路由入口
  styles/         # 全局样式
  utils/          # 数据、路径、日期等工具
```

## 内容集合

内容集合在 `src/content.config.ts` 中定义：

```ts
export const collections = {
  home,
  blogs,
  projects,
}
```

对应 schema 在 `src/content/schema.ts`：

- `postSchema`：博客 frontmatter
- `projectSchema`：项目 JSON 数据

## 博客链路

博客内容放在：

```text
src/content/blogs/
```

列表页链路：

```text
src/pages/blogs/index.mdx
  -> ListView.astro
  -> getGroupedPostsByYear('blogs')
  -> src/content/blogs/**/*.md(x)
```

详情页链路：

```text
src/pages/blogs/[...slug].astro
  -> getFilteredPosts('blogs')
  -> RenderPost.astro
  -> StandardLayout.astro
```

生产构建会过滤 `draft: true` 的文章。

## 项目链路

项目数据放在：

```text
src/content/projects/data.json
```

页面链路：

```text
src/pages/projects.mdx
  -> GroupView.astro
  -> getCollection('projects')
  -> GroupItem.astro
```

当前 `data.json` 可以保持空数组：

```json
[]
```

有项目时追加对象即可。

## 搜索

搜索由 Pagefind 提供，只收录博客正文。

配置位置：

```ts
FEATURES.search = [
  true,
  {
    includes: ['blogs'],
    filter: false,
    navHighlight: true,
    batchLoadSize: [true, 5],
    maxItemsPerPage: [true, 3],
  },
]
```

开发环境下搜索面板使用模拟数据；真实索引会在 `pnpm build` 后生成到 `dist/pagefind/`。

## 日常维护

新增博客：

1. 复制 `src/content/blogs/template.md`
2. 修改文件名、标题、描述、标签和日期
3. 发布时设置 `draft: false`

新增项目：

1. 编辑 `src/content/projects/data.json`
2. 添加 `id`、`link`、`desc`、`icon`、`category`
3. 运行 `pnpm check` 确认 JSON 符合 schema

修改首页：

```text
src/content/home/index.md
```

修改导航、站点名、社交链接：

```text
src/config.ts
```
