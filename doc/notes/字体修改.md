# 字体修改指南

这份文档用于说明当前项目里如何接入本地字体，以及如何修改某个具体区域的字体、字号和样式。

## 推荐做法

- 如果你下载的是本地 `ttf` 字体，最稳妥的方式是：`public/fonts/ + @font-face`
- 这样以后不只是改 logo，想改首页标题、正文、博客内容、按钮文案，都可以复用
- 字体文件放在 `public/` 下时，引用路径统一写成站点根路径，例如 `/fonts/xxx.ttf`

## 一步一步操作

### 1. 放字体文件

建议在项目里新建目录：

```text
public/fonts/
```

例如把字体文件放成：

```text
public/fonts/MyChineseFont.ttf
```

### 2. 在全局 CSS 注册字体

打开 `src/styles/main.css`，在文件顶部加入：

```css
@font-face {
  font-family: 'MyChineseFont';
  src: url('/fonts/MyChineseFont.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}
```

说明：

- `font-family` 是你后续在样式里要使用的字体名
- `src` 要和实际字体文件路径一致
- `font-display: swap;` 建议保留，避免页面长时间空白

## 通用规则

如果你想改“某个地方”的字体，建议按下面顺序做：

1. 先找到这个文字是哪个组件或布局渲染的
2. 找到它对应的 class、标签或容器
3. 只在那个选择器上改 `font-family`
4. 如果还需要视觉调整，再同时改 `font-size`、`line-height`、`font-weight`

例如：

```css
.some-title {
  font-family: 'MyChineseFont', sans-serif;
  font-size: 2.5rem;
  line-height: 1.1;
}
```

## 常见场景

### 1. 修改左上角站点标题 `WutongYu`

位置：

- 文件：`src/components/widgets/LogoButton.astro`
- 选择器：`.logo-text`

当前样式类似：

```css
.logo-text {
  font-family: 'Great Vibes', cursive;
  font-size: 1.65rem;
  line-height: 1;
  color: var(--c-text);
}
```

如果要改成你的本地字体，可以写成：

```css
.logo-text {
  font-family: 'MyChineseFont', 'Great Vibes', cursive;
  font-size: 1.65rem;
  line-height: 1;
  color: var(--c-text);
}
```

说明：

- 优先使用本地字体
- 如果字体加载失败，再退回 `Great Vibes`

### 2. 修改首页大标题 `Wutong Yu`

这里说的是首页中 `My Blog Overview` 上方的大标题，不是左上角导航里的 logo。

标题来源：

- `src/pages/index.mdx` 中的 frontmatter：

```md
title: 'Wutong Yu'
subtitle: My Blog Overview
```

渲染位置：

- `src/layouts/StandardLayout.astro`

当前结构类似：

```astro
<header class={`prose mx-auto mb-16${isCentered ? ' text-center' : ''}`}>
  {title && <h1>{title}</h1>}
  {subtitle && <p class="mt--4! op-50 italic">{subtitle}</p>}
  <slot name="head" />
</header>
```

如果你想单独控制这个大标题，推荐先给 `h1` 加一个类：

```astro
{title && <h1 class="page-title">{title}</h1>}
```

然后在 `src/styles/main.css` 中添加样式：

```css
.prose h1.page-title {
  font-family: 'MyChineseFont', sans-serif;
  font-size: 3rem;
  line-height: 1.1;
}
```

为什么推荐写成 `.prose h1.page-title`：

- 这个项目里 `src/styles/prose.css` 已经给 `.prose h1` 设置了默认字号
- 写成更具体的选择器，能更稳定地覆盖原有规则

如果你还想兼顾手机端，可以再补一段：

```css
@media (max-width: 767.9px) {
  .prose h1.page-title {
    font-size: 2.4rem;
  }
}
```

### 3. 修改全站正文

推荐直接在 `src/styles/main.css` 加：

```css
body {
  font-family: 'MyChineseFont', sans-serif;
}
```

注意：

- 这会影响全站大部分文字
- 不建议在一开始就直接全站替换，最好先局部验证字体效果

### 4. 修改博客正文

文件：

- `src/styles/prose.css`

可以这样写：

```css
.prose {
  font-family: 'MyChineseFont', sans-serif;
}
```

### 5. 修改博客标题

仍然建议写在 `src/styles/prose.css`：

```css
.prose h1,
.prose h2,
.prose h3,
.prose h4 {
  font-family: 'MyChineseFont', sans-serif;
}
```

### 6. 修改导航菜单

做法：

- 去 `src/components/nav/` 下找到对应组件
- 找到菜单文字对应的 class
- 对那个 class 单独设置 `font-family`

## 字号改哪里

很多时候你不只是想换字体，还想“让它更大一些”。这时就在同一个选择器里加 `font-size`。

例如首页大标题：

```css
.prose h1.page-title {
  font-family: 'MyChineseFont', sans-serif;
  font-size: 3.2rem;
  line-height: 1.1;
}
```

常见参考值：

- `2.5rem`：略微变大
- `3rem`：明显变大
- `3.5rem`：视觉重点更强

## 当前项目里常见修改入口

- 首页大标题：`src/layouts/StandardLayout.astro`

- 全局样式入口：`src/styles/main.css`
- 博客正文与标题：`src/styles/prose.css`
- 导航相关组件：`src/components/nav/`
