Hello

This streamlined Astro site keeps the original Antfu-style visual language while focusing on three core destinations:

- [Blog](/blog/) for long-form writing and searchable post content
- [Projects](/projects/) for grouped project showcases
- [GitHub](https://github.com/lin-stephanie/astro-antfustyle-theme) as an external profile entry in the top navigation

**What Stays**

- Responsive layout and keyboard-accessible navigation
- Light and dark theme switching
- Blog search powered by Pagefind
- Markdown and MDX-based content authoring
- Animated backgrounds and the original minimalist reading experience

**Start Exploring**

- Browse the latest posts on [the blog index](/blog/)
- Open [the project gallery](/projects/)
- Use the search button in the navigation to search blog posts only

The result is a smaller, easier-to-maintain starter while preserving the original look and feel where it matters most.
